vsanalog
========
A VapourSynth plugin providing sources and filters for working with digitized
analog video and signals.

The code is managed on the
`JustinTArthur/vapoursynth-analog project on GitHub <https://github.com/JustinTArthur/vapoursynth-analog>`_.
